# f1-grid-sorter
F1 Grid Sorter: Rank your favorite drivers and share the results.
### Features:
 *The user selects one of two drivers until a complete ranking is formed.
 *Once the sorting is complete, a list is generated with medals for the top 3 pilots.
 *A futuristic, telemetry-style interface.
 *The ability to copy the full list of results with a single click to share with friends.
### Technologies:
  **HTML5/CSS3:** Layout using Flexbox and custom animations.
  **JavaScript (Vanilla):** Sorting logic, array handling and clipboard copy function
